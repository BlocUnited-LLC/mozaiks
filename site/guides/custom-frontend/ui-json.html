ui.json

Configure the header, profile dropdown, notifications button, footer links, and chat mode styling. No colors or font definitions belong here.

📁

File location: brands/public/ui.json — served at /ui.json

Full example

brands/public/ui.json
{
  "chat": {
    "modes": {
      "ask":      { "tint": "#06b6d4", "label": "Ask" },
      "workflow": { "tint": "#8b5cf6", "label": "Workflow" }
    },
    "bubbleRadius": "18px"
  },

  "header": {
    "logo": {
      "src":      "mozaik_logo.svg",
      "wordmark": "mozaik.png",
      "alt":      "App logo",
      "href":     "https://yourapp.com"
    },
    "actions": [
      {
        "id":      "discover",
        "label":   "Discover",
        "icon":    "sparkle.svg",
        "variant": "gradient",
        "visible": true
      }
    ]
  },

  "profile": {
    "icon":         "profile.svg",
    "show":         true,
    "defaultLabel": "Commander",
    "sublabel":     "Mission Control",
    "menu": [
      {
        "id":     "profile-settings",
        "label":  "Profile Settings",
        "icon":   "settings.svg",
        "action": "navigate",
        "href":   "/profile"
      },
      {
        "id":     "preferences",
        "label":  "Preferences",
        "icon":   "settings.svg",
        "action": "navigate",
        "href":   "/preferences"
      },
      {
        "id":   "divider-before-signout",

      },
      {
        "id":      "signout",

chat

FieldTypeDescription
chat.modes.ask.tint string Accent color for the "Ask" mode pill/indicator. Any valid CSS color.
chat.modes.ask.label string Human-readable label for Ask mode (e.g. "Ask", "Chat").
chat.modes.workflow.tint string Accent color for the "Workflow" mode indicator.
chat.modes.workflow.label string Human-readable label for Workflow mode.
chat.bubbleRadius string optional CSS border-radius for chat message bubbles. Default: "18px".

header

FieldTypeDescription
header.logo.src string Logo icon filename — resolved from /assets/. Example: "logo.svg"
header.logo.wordmark string optional Wordmark image filename shown next to or instead of the icon in expanded layout.
header.logo.alt string Alt text for the logo image (accessibility).
header.logo.href string optional URL the logo links to. Defaults to "/" if omitted.

header.actions

An ordered array of action buttons rendered in the right side of the header toolbar. Each visible action with a valid icon renders as a button.

FieldTypeDescription
[*].id string required Unique identifier. Passed to onAction(id, item) when clicked.
[*].label string Button label text shown next to the icon.
[*].icon string required Icon filename — must include a dot (e.g. "sparkle.svg"). Token strings like "sparkle" are not supported and log a warning.
[*].variant string optional "gradient" renders a cyan–violet gradient button. Omit for a plain ghost button.
[*].visible boolean optional Set to false to hide without removing. Defaults to true.

profile

FieldTypeDescription
profile.icon string Fallback avatar icon filename when the user has no photo (e.g. "profile.svg").
profile.show boolean Set to false to hide the profile button entirely.
profile.defaultLabel string Display name shown when no authenticated user is present (e.g. "Guest").
profile.sublabel string optional Secondary line (plan tier, role). Shown under the name in the dropdown header.
profile.menu array Ordered array of menu items. See below.

profile.menu

Three item types: action items, dividers. All rendered in declaration order.

Action item

Navigate item
{
  "id":     "profile-settings",
  "label":  "Profile Settings",
  "icon":   "settings.svg",
  "action": "navigate",
  "href":   "/profile"
}
Destructive action (danger variant)
{
  "id":      "signout",
  "label":   "Sign Out",
  "icon":    "logout.svg",
  "action":  "signout",
  "variant": "danger"
}
Divider
{
  "id":   "divider-1",
  "type": "divider"
}
FieldTypeDescription
id string required Unique key (used as React key).
type string optional Set to "divider" to render a horizontal rule. All other fields ignored.
label string required Visible menu label text.
icon string optional Icon filename (must contain a dot). See Step 4 — Assets.
action string Passed to onAction(action, item). Built-in: "navigate", "signout". Custom values are forwarded as-is.
href string optional Route path for navigate actions. Passed inside the item object to onAction.
variant string optional "danger" renders in the error/red color. Use for destructive actions.

notifications

FieldTypeDescription
notifications.icon string Icon filename for the notification button (e.g. "notifications.svg").
notifications.show boolean Set to false to hide the notification button entirely.
notifications.emptyText string optional Text rendered in the notification panel when there are no items.

footer

FieldTypeDescription
footer.links array Array of { "label": string, "href": string } rendered as footer links.
footer.visible boolean Set false to hide the footer bar entirely.
footer.poweredBy string | null optional "Powered by …" attribution text. Set null to omit.
