Mozaiks Developer Docs

Build something yours.

Mozaiks gives you a fully brandable AI chat runtime. This documentation walks you through customizing every part of your frontend — colors, menus, icons, and backend wiring.

ℹ️

Everything in the UI shell is driven by two JSON config files. No code changes needed for most customizations.

Customizing Your Frontend

Step 1 — Overview & Quick Start

Understand the two-file config system and get your dev environment running in minutes.

Step 2 — brand.json

Set your colors, typography, shadows, and asset filenames. Full field reference included.

Step 3 — ui.json

Icon format requirements, file placement, and how the ActionIcon component works.

Repository structure

Template layout
packages/frontend/chat-ui/
  template/
    vite.config.js        # publicDir: brands/public/
    brands/
      public/             # Served at /
        brand.json        # Visual identity
        ui.json           # Chrome config
        assets/           # Icons + images

The config split

brand.json

Colors · typography · shadows · asset filenames.
Controls what the UI looks like.

ui.json

Header actions · profile menu · notifications · footer.
Controls what appears in the shell.
